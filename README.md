# mines

Set-based implementation of minesweeper in Python

## Usage

```shell
python3 demo.py --help
```

* `q` - quit
* `Enter` - reveal
* `Space` - flag/chord
* `Arrow keys` - move cursor

## Documentation

For a mathematical explanation of the data structures and algorithms, see [math.pdf](docs/math.pdf) which is the rendered output of [math.tex](docs/math.tex).
